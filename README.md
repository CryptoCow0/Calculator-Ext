# Calculator-Ext
A calculator extension for Chrome

## Supported Features

### Operators
| Operator | Name                  |
|----------|-----------------------|
|    \+    |  Add                  |
|    \-    |  Subtract             |
|    \*    |  Multiply             |
|    /     |  Divide               |
|    %     |  Modulus              |
|    ^     |  Exponent             |
|    !     |  Factorial            |
|    E     |  Scientific Notation  |

### Constants
- pi
- e

### Functions
- abs (absolute value)
- rand (random number generator)
- sqrt (square root)
- cbrt (cube root)
- floor (round down)
- round (round to nearest whole number)
- ceil (round up)
- sin / asin
- cos / acos
- tan / atan
- log (logarithm)
- ln (natural logarithm)

### Other
- Supports both degree (deg) and radian (rad) modes
