# Calculator-Ext
A calculator extension for Chrome

## Supported Features

### Constants
- pi
- e

### Functions
- abs (absolute value)
- rand (random number generator)
- sqrt (square root)
- cbrt (cube root)
- floor (round down)
- round (round to nearest whole number)
- ceil (round up)
- sin
- cos
- tan
- log (logarithm)
- ln (natural logarithm)
